Snowy Sweep

by Mikala Spencer




How far can you make it before you crash into a snow pillar?

Use the arrow keys to navigate the snowy landscape:
- Press the "UP" arrow key to jump
- Press the "LEFT" arrow key to move left
- Press the "RIGHT" arrow key to move right
- Press the "DOWN" arrow key to move down

Collect the orange "balls of warmth" for extra points.

Select the "Story" button to learn about game.

Select the "Length" button to set how long you would like to play the game.
Otherwise, the game will go on until you hit an obstacle.
